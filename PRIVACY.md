# Privacy Policy

**Last updated:** April 2, 2026

## Overview

LOD Manager is a free and open-source application for managing Loglan Online Dictionary (LOD) format databases. This Privacy Policy explains how the application handles your data.

## Data Collection

**LOD Manager does not collect, store, or transmit any personal data.**

The application operates entirely offline and does not:

- Collect personal information
- Use analytics or tracking services
- Send usage data to any server
- Access your contacts, photos, or other personal data
- Use advertising libraries
- Integrate with third-party services that collect data

## Data Stored Locally

The application stores data only on your device:

- **Dictionary databases** (`.db` files) that you explicitly open or import
- **Application preferences** stored in the platform's standard configuration directory

This data never leaves your device unless you manually share or export it.

## Permissions

### Desktop (Windows/macOS/Linux)

- File system access — only for files you explicitly select via file dialogs

### Android

- Storage access — only through the system file picker for opening dictionary files

### iOS

- File access — only through the system document picker for opening dictionary files

## Third-Party Services

LOD Manager uses the following open-source libraries, none of which collect personal data:

- **Tauri** — application framework (tauri.app)
- **Svelte** — frontend framework (svelte.dev)
- **rusqlite** — SQLite bindings for Rust (github.com/rusqlite/rusqlite)

## Children's Privacy

The application does not knowingly collect any information from anyone. It is suitable for users of all ages.

## Changes to This Policy

Any changes to this Privacy Policy will be posted here with an updated revision date.
