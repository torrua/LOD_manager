## Directory Purposes

**`src/` (Frontend):**

- Contains: Svelte 5 components, TypeScript, global styles
- Key files: `App.svelte`, `main.ts`, `types.ts`
- Components in `lib/components/`

**`src-tauri/src/` (Backend):**

- Contains: Rust source code for Tauri commands
- Key files: `lib.rs`, `db.rs`, `models.rs`, `import.rs`, `export.rs`

**`tables/`:**

- Contains: SQLite databases and LOD import files
- Generated by app or provided for import

**`dist/`:**

- Contains: Built frontend assets (generated by `npm run build`)
- Committed: Yes (for releases)

## Key File Locations

**Entry Points:**

- `src/main.ts`: Frontend entry - mounts Svelte app
- `src-tauri/src/lib.rs`: Backend entry - defines Tauri commands

**Configuration:**

- `package.json`: npm scripts, dependencies
- `tsconfig.json`: TypeScript configuration
- `vite.config.ts`: Vite build configuration
- `src-tauri/Cargo.toml`: Rust dependencies
- `src-tauri/tauri.conf.json`: Tauri app configuration
- `.prettierrc`, `eslint.config.js`: Code formatting/linting

**Core Logic:**

- `src-tauri/src/lib.rs`: All Tauri commands (~27 commands)
- `src-tauri/src/db.rs`: Database schema, queries, FTS
- `src-tauri/src/models.rs`: Rust structs for serialization

**Testing:**

- `src-tauri/src/lib.rs`: Contains inline tests under `#[cfg(test)]`

## Naming Conventions

**Svelte Components:**

- PascalCase: `WordDetail.svelte`, `EventForm.svelte`

**TypeScript:**

- camelCase: `store.svelte.ts`, `text.ts`

**Rust Modules:**

- snake_case: `db.rs`, `import.rs`, `export.rs`

**Rust Types:**

- PascalCase: `WordListItem`, `SaveWord`, `ELResult`

## Where to Add New Code

**New Tauri Command:**

- Location: `src-tauri/src/lib.rs`
- Add function with `#[tauri::command]` attribute
- Add to `invoke_handler` in `run()` function

**New Database Operation:**

- Location: `src-tauri/src/db.rs`
- Follow existing function patterns (e.g., `save_word`, `list_words`)

**New Frontend Component:**

- Location: `src/lib/components/`
- Use existing components as templates

**New Type Definition:**

- Frontend: `src/types.ts`
- Backend: `src-tauri/src/models.rs`
